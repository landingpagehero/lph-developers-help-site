Local dev tool

The dev tool below replicates some of the LPH server functionality: it compiles files named .scss (written in SCSS) to .css (normal CSS), and compiles files named .js6 (written in ES6) to .js (ES5). These changes are done on the server automatically, so you should use this tool locally too. It can be thought of as a quick Grunt/Gulp replacement.

Linux user? Download the tool with:
sudo curl developers.lph.megathings.co.uk/tool-for-linux -o /usr/bin/lph && sudo chmod +x /usr/bin/lph

Mac user? Download the tool with:
sudo curl developers.lph.megathings.co.uk/tool-for-mac -o /usr/bin/lph && sudo chmod +x /usr/bin/lph

Now just run the command lph in any directory and files in the directories `./styles/` and `./scripts/` will be compiled and watched for changes.

The tool's source code is available on GitHub.
